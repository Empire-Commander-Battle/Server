# Setup
https://www.fsegames.eu/forum/index.php?topic=11836.0

## Linux
### Enter shell
```bash
./guix-shell.sh
```
### Building
```bash
cd Source/Sourcecode/mm dev - MS/
./build_module.sh
```

### Running server
```
./run-server.sh
```

### Connect to server repl
```
python3 remote-shell.py
```
